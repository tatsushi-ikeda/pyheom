# pyheom

**pyheom** is a Python package for simulating open quantum system dynamics using the
Hierarchical Equations of Motion (HEOM) and the Redfield master equation.

Based on: Ikeda & Scholes, *J. Chem. Phys.* **152**, 204101 (2020), https://doi.org/10.1063/5.0007327 (arXiv: https://arxiv.org/abs/2003.06134).
